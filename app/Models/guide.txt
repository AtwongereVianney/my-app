# Laravel Model Display Guide

This guide provides step-by-step instructions for displaying any Laravel model in the browser with full CRUD functionality.

## Prerequisites

- Laravel project setup
- Model created with proper `$fillable` properties
- Database migration run

## Step-by-Step Process

### 1. Create Controller

```bash
php artisan make:controller ModelNameController
```

**Edit the controller file:**

```php
<?php

namespace App\Http\Controllers;

use App\Models\ModelName;
use Illuminate\Http\Request;

class ModelNameController extends Controller
{
    public function index()
    {
        $modelnames = ModelName::all();
        return view('modelnames.index', compact('modelnames'));
    }

    public function show($id)
    {
        $modelname = ModelName::findOrFail($id);
        return view('modelnames.show', compact('modelname'));
    }

    public function create()
    {
        return view('modelnames.create');
    }

    public function store(Request $request)
    {
        $request->validate([
            // Add validation rules for your model fields
            'field1' => 'required|string|max:255',
            'field2' => 'required|email|unique:table_name',
        ]);

        ModelName::create($request->all());
        return redirect()->route('modelnames.index')->with('success', 'Record created successfully!');
    }

    public function edit($id)
    {
        $modelname = ModelName::findOrFail($id);
        return view('modelnames.edit', compact('modelname'));
    }

    public function update(Request $request, $id)
    {
        $modelname = ModelName::findOrFail($id);
        
        $request->validate([
            // Add validation rules (include unique rule exception for updates)
            'field2' => 'required|email|unique:table_name,field2,' . $modelname->id,
        ]);

        $modelname->update($request->all());
        return redirect()->route('modelnames.index')->with('success', 'Record updated successfully!');
    }

    public function destroy($id)
    {
        $modelname = ModelName::findOrFail($id);
        $modelname->delete();
        return redirect()->route('modelnames.index')->with('success', 'Record deleted successfully!');
    }
}
```

### 2. Add Routes

**In `routes/web.php`:**

```php
use App\Http\Controllers\ModelNameController;

Route::resource('modelnames', ModelNameController::class);
```

### 3. Create View Directory Structure

Create directory: `resources/views/modelnames/`

### 4. Create View Files

                        <tr>

                    </thead>

                                <td>{{ $item->id }}</td>

                                        <a href="{{ route('modelnames.show', $item->id) }}"

                                                <i class="fas fa-trash"></i>
                                            </button>

                                    </div>

## Notes

- Replace `ModelName` with your actual model name
- Replace `modelnames` with your preferred route names (usually plural, lowercase)
- Replace `field1`, `field2` etc. with your actual model fields
- Update validation rules based on your model requirements
- Customize icons and styling as needed
- Ensure your model has proper `$fillable` properties defined
